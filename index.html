    Notenverwaltung

    Die Notenverwaltungssoftware für Schulen und Lehrer. Kostenlos und Open-Source jetzt verfügbar.

    C#

    Die Software wurde in C# zusammen mit dem Windows Presentation Foundation(WPF) geschrieben. Es wurde auf neuste Frameworks sowie dem MVVM-Standard gesetzt.

    Interaktive Oberfläche

    Durch die interaktive Oberfläche ist die Anwendung für Lehrkräfte nochmal um einiges einfacher und übersichtlicher.

    GitHub

    Der ganze Programmcode steht Open-Source auf GitHub zur Verfügung.
